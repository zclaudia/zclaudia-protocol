# @zclaudia/protocol

Shared gateway protocol and notification wire types for ZClaudia packages.

## Install

```bash
npm install @zclaudia/protocol
```

## Usage

```ts
import type { BackendClientMessage } from '@zclaudia/protocol/gateway';
import { DEFAULT_NOTIFICATION_CONFIG } from '@zclaudia/protocol/notifications';
```

## Exports

- `@zclaudia/protocol`
- `@zclaudia/protocol/gateway`
- `@zclaudia/protocol/notifications`
